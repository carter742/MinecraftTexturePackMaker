<h1>Usage:</h1>
<p>This program is design to be a simple and fast way to duplicate photos onto all minecraft textures while persevering the general colour of the original textures.</p>

<p>In order to use this program you first need to have go installed. (Click <a href="https://go.dev/doc/install">here</a> for more detials on how to install).</p>

<p>After that open the project directory and dropany photos you want to use as a texture into the "pictures" directory.</p>

<p>You can now run the program. It will prompt you to name the pack. After doing so the pack will be generated within the same directory.</p>

<p>Enjoy! :)</p>
